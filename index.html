About

ThingsToDoPRO is an independently developed side project that is made with Microsoft Dotnet, and is being worked on with many new features. For any new suggestions, open a new issue on Github.

Install Tutorial

How to install ThingsToDoPRO:

  1. Download the installer file from GitHub
  2. Run the installer as usual, adjusting settings for yourself
  3. When it is installed, make sure (via compatibility settings) that it is being run as an administrator (the app will not work if you ignore this step)
  4. Run the program and enjoy! It will save automatically and load your to-do list.
